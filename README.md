# Binary Calculator 

## What you need
- Arduino Uno or SparkFun RedBoard
- 4 yellow and 4 red LEDs
- 9 jumper wires
- 8 330 Ohm resistors 
- 1/2 breadboard

## What it is supposed to do
Given 2 integers and a math operator, show the result as binary by lighting up LEDs.
